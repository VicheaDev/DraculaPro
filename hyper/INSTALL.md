### [Hyper](https://hyper.is/)

#### Install

1. Copy theme folder
2. Paste theme folder into `~/.hyper_plugins/local`
3. Open `~/.hyper.js` file
4. Add `dracula-pro` to the local plugins list

    localPlugins: [
    	"dracula-pro"
    ]

#### Activating theme

1.  Start Hyper
2.  Go to `View -> Full Reload`
